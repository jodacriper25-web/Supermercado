# Manual de Usuario y Documentación Técnica

## Manual de Usuario

### Acceso y Registro
- Los clientes pueden navegar el catálogo sin registrarse.
- Para realizar un pedido, deben crear una cuenta o iniciar sesión.
- El administrador accede a la gestión desde /admin o /dashboard-admin.

### Catálogo y Búsqueda
- Usa la barra de búsqueda o el menú de categorías para filtrar productos.
- Haz clic en "Agregar al carrito" para seleccionar productos.

### Pedidos
- Accede al carrito y sigue el proceso de checkout.
- Ingresa dirección y confirma el pedido.
- Puedes ver el historial de pedidos en tu panel.

### Contacto
- Usa el botón flotante de WhatsApp para consultas rápidas.

## Documentación Técnica

### Estructura del Proyecto
- Django 4.x, Bootstrap 5, Chart.js
- Apps: core (lógica principal), supermercado (configuración)

### Modelos principales
- Categoria, Producto, Pedido, DetallePedido, Wishlist

### Funcionalidades clave
- Importación masiva de productos vía XML
- Panel de administración personalizado (/dashboard-admin)
- Alertas de stock bajo
- Gráficos de ventas
- Pruebas unitarias en core/tests.py

# Imágenes de Productos

Aquí puedes agregar las imágenes PNG o JPG de los productos.

## Estructura de carpetas recomendada:

```
products/
├── consumo/
├── limpieza-y-hogar/
├── bebidas/
├── congelados/
└── confiteria/
```

## Instrucciones:

1. Crea subcarpetas para cada categoría (opcional)
2. Agrega tus imágenes con nombres descriptivos (ej: `tips_banio_pastilla_90g.jpg`)
3. Los formatos soportados son: PNG, JPG, JPEG, WebP

**Nota:** Las imágenes se mostrarán automáticamente en los productos según la URL que tengas en tu base de datos.

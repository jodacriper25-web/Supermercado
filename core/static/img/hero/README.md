Coloca aquí las imágenes que deseas mostrar en el carrusel (hero).
Nombres y formato recomendados: hero1.jpg, hero2.jpg, hero3.jpg (jpg/png/svg).

Puedes definir duraciones por archivo (segundos) editando `intervals.json`.
Ejemplo: { "hero1.jpg": 30, "hero2.jpg": 10 }

Para usar las imágenes que enviaste, copia los archivos dentro de esta carpeta.
